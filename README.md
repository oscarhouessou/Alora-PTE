Alora PTE is a modern, full-featured practice application for the Pearson Test of English (PTE). Designed with a premium light-themed aesthetic, it provides a seamless and interactive environment for students to master all sections of the exam.

## ‚ú® Features

### üé® Premium Design
- **Light Theme Redesign**: A modern, airy interface featuring soft mesh gradients and professional typography.
- **Glassmorphism**: Elegant white-glass cards with subtle shadows and high-end visual feedback.
- **Dynamic Stats**: Real-time progress tracking and category-weighted performance analytics.

| Category | Exercises |
| :--- | :--- |
| **Speaking** | Read Aloud, Repeat Sentence, Describe Image, Re-Tell Lecture |
| **Writing** | Summarize Written Text, Write Essay |
| **Reading** | Fill in the Blanks (Dropdown), Reorder Paragraphs |
| **Listening** | Summarize Spoken Text, Fill in the Blanks, Highlight Correct Summary, Multiple Choice |

### ü§ñ Intelligent Features
- **AI Scoring**: Integrated with Groq AI (Llama 3.3 70B) for real-time feedback on speaking and writing tasks.
- **Voice Recognition**: Interactive audio recording and transcription for speaking exercises.
- **Smart Timer**: Section-specific preparation and response timers for realistic exam simulation.

## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/oscarhouessou/Alora-PTE.git
   cd Alora-PTE
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env` file in the root directory and add your Groq API key:
   ```env
   VITE_GROQ_API_KEY=your_groq_api_key_here
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
